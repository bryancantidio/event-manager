# 📅 Sistema de Eventos

Sistema simples de gerenciamento de eventos desenvolvido com HTML, CSS e JavaScript.

---

## 🚀 Funcionalidades

- Cadastro de eventos (título, local, vagas, preço e data)
- Marcar evento como ativo ou inativo
- Listagem dinâmica dos eventos cadastrados
- Exibição de estatísticas:
  - Total de eventos
  - Eventos ativos
  - Total de arrecadação possível

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

## 📌 Observações

Projeto desenvolvido com fins de aprendizado, focado em manipulação de DOM, arrays de objetos e funções como map, filter e reduce.
